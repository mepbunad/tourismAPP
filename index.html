Navegación basada en mapa

Mapa interactivo para explorar rutas y experiencias

Este bloque ya deja listo el uso de Leaflet.js con OpenStreetMap. Solo debes conservar el contenedor del mapa y actualizar las coordenadas, textos y enlaces de cada marcador según tus rutas reales. Si prefieres Google Maps, puedes reemplazar este bloque por su SDK y reutilizar la misma estructura de tarjetas y botones.

Cómo usar la API / librería del mapa

  1. 1. Mantén incluido Leaflet desde CDN en el <head> y antes del cierre del <body>.
  2. 2. En script.js define un arreglo con nombre, categoría, coordenadas, descripción y ancla de cada experiencia.
  3. 3. Cada marcador abre un popup con resumen y botón para ir a la subsección correspondiente.
  4. 4. Si luego quieres rutas reales paso a paso, puedes integrar Leaflet Routing Machine o migrar a Google Maps Directions API.
